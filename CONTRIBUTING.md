# Contributing to Pulse

Pulse'a katkıda bulunmak istediğiniz için teşekkürler! Bu proje, topluluğun desteğiyle daha güvenli bir internet vizyonuna hizmet etmektedir.

## Nasıl Katkıda Bulunabilirim?

### 1. Hata Bildirimi (Bug Reports)
Eğer bir hata bulursanız, lütfen GitHub Issues üzerinden bildirin. Bildiriminizde şunları eklemeyi unutmayın:
- Kullandığınız işletim sistemi ve Docker versiyonu.
- Hatanın oluşma adımları.
- Hata logları.

### 2. Özellik Önerileri (Feature Requests)
Yeni bir fikir mi var? "Feature request" etiketiyle bir issue açarak toplulukla paylaşın.

### 3. Pull Request (PR) Gönderme
Kod yazarak katkıda bulunmak isterseniz:
1. Projeyi **Fork** edin.
2. Yeni bir feature branch açın (`git checkout -b feature/harika-ozellik`).
3. Değişikliklerinizi yapın ve commit mesajlarınızı açıklayıcı tutun.
4. Branch'inizi push edin (`git push origin feature/harika-ozellik`).
5. Bir **Pull Request** açın.

## Kod Standartları
- Python kodları için **PEP 8** standartlarına uyun.
- Mümkünse asenkron yapıyı bozmamaya özen gösterin.
- Yeni eklediğiniz fonksiyonlar için docstring eklemeyi unutmayın.

## İletişim
Sorularınız için GitHub Issues kısmını kullanabilir veya proje sahiplerine ulaşabilirsiniz.
